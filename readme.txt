== Description ==

A simple WordPress heatmap plugin to monitoring your user's behaviour, detect and report abnormal user activities. This plugin allows you to install UXsniff tracking code to your wordpress without editing your theme. 

The plugin also shows basic statistic about your website. Some basic features such as Heatmaps, Clickmaps, Session Recordings and Rage alerts are available within the plugin. Login to UXsniff for advanced features such as user journey, feedback widgets, on-site surveys, broken link scanner, site audit and advanced reports.

**Plugin Features**

* Insert UXsniff tracking code
* Report abnormal user activities
* Heatmaps
* Clickmaps
* Session recordings
* Rage alerts

**Advanced Features (available on uxsniff.com for free)**

* Real-time monitoring
* GPT summaries for session recordings
* Wayback machine for heatmaps (time travel to old designs)
* Feedback widgets
* On-site surveys
* LinkGuard - broken links scanner
* Site audit - scan your site for UX and SEO issues

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

After active plugin go to UX Sniff->Settings to setup your tracker.
